# Open Seseame #

## Summary ##

My password search engine.

## Dependence ##
### 1.WebServer ###
any

### 2.PHP ###
* Version：5.3+
* Extension：zlib

## Install ##

* Step 1:
```
cd $document_root
git clone git@bitbucket.org:liuxd/open-sesame.git
cd open-sesame/res/ini/
cp web.ini.sample web.ini
```

* Step 2:
```
Configure the [db].Make sure the path is writable.
```
