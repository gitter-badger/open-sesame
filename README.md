## 需求
> 最基本的PHP环境即可（PHP5.4+）

## 安装

```
git clone git@github.com:open-sesame/open-sesame-web.git
cd open-sesame-web/src
php -S localhost:8000
```
可以访问`localhost:8000`了。

> 注意：根据自己需求修改config.ini，并放到src下。
